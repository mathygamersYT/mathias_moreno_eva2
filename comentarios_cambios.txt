Comentarios de cambios aplicados:

1. El carrusel manual fue movido desde la seccion Hardware hacia el bloque principal "Bienvenido a IT Progs", para que sea visible inmediatamente al cargar la pagina.

2. La seccion Hardware fue eliminada a solicitud del usuario para evitar texto duplicado y dejar la pagina mas limpia.

3. Se agrego un servidor local en server.js. Este servidor entrega index.html, styles.css, script.js, imagenes y tambien permite leer/escribir usuarios.json mediante la ruta /api/usuarios.

4. El registro y login se mantienen como ventanas emergentes desde el navbar. Al registrar un usuario, script.js envia los datos al servidor local para que usuarios.json se actualice automaticamente.

5. El servidor local usa Cache-Control: no-store para que los cambios realizados en los archivos del proyecto se vean reflejados al recargar el navegador.

6. Se conserva JavaScript puro para la logica del carrusel, validacion de formularios, clase Usuario, arreglo usuariosInscritos y funciones validarDatos(), cambiarImagen() y actualizarDOM().
